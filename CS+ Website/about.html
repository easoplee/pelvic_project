Bone Model Project

ABOUT

This project is affiliated with the Duke University Department of Computer Science and Department of Radiology. In collaboration with the Mazurowski Lab, Easop Lee, Ashka Shah, and Mohammed Elmouzi have started working on the bone project. The goal of this project is to create a 3D model of human bones using 3D MRI images. We have started using bone segmentation algorithms to identify different bones in pelvic region MRI scans. Then, we use that data to create 3D models of the bones visible in the pelvic region.
