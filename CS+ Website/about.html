Bone Model Project

ABOUT

This project is affiliated with the Duke University Department of Computer Science and Department of Radiology. In collaboration with the Mazurowski Lab, Easop Lee, Ashka Shah, and Mohammed Elmouzi have started working of the bone project. The goal of this project is to create a 3D model of the human body using 3D MRI images. We have started using bone segmentation algorithms to identify different bones in pelvic region MRI scans.
